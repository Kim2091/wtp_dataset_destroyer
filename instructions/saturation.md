```json 
  {
    "type": "saturation",
    "rand": [0.5, 1.0],
    "probability": 0.5
  }
```
`*` = optional parameters

- `rand` - The range from which a random value will be picked to control saturation. `[low,high]`
- `probability`* - The chance of applying (e.g. 0.5 = 50% chance of being applied)
